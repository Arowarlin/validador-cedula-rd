Verificador de Cédula

Introduzca el número de cédula (con o sin guiones) para verificar su validez estructural.

Formato: 11 dígitos (Ej: 40212345678 o 402-1234567-8)

📋 Ejemplos / Pruebas

  • ✓ 00113918205 — ejemplo válido
  • ✓ 40227457803 — ejemplo válido
  • ℹ El validador muestra los cálculos para ayudarte a entender el algoritmo

📌 Notas importantes

  • Se valida la estructura (11 dígitos) y el dígito verificador con algoritmo Módulo 10 (Luhn).
  • Este validador prueba múltiples variantes del algoritmo para máxima compatibilidad.
  • El validador no consulta la base de datos de la JCE; solo comprueba la regla estructural.
  • Se muestran los detalles de cálculo para propósitos educativos.
